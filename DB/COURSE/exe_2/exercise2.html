SQL

  • Structure Query Language, ou Linguagem de Consulta Estruturada é o termo completo de SQL;
  • É a linguagem de pesquisa declarada padrão para banco de dados relacional;
  • Podemos afirmar que é uma linguagem de programação;
  • Alguns dos principais sistemas que utilizam essa tecnologia são: MYSQL, PostgressSQL, Oracle, MariaDB e SQL Server;
  • A linguagem SQL apresenta uma organização estrutural muito bem definida que divide seus comando em 5 subconjuntos diferentes que são: DQL, DML, DDL, DCL, E DTL;
  • Cada um desses subconjuntos apresenta objetivos específicos;

DQL: Linguagem de consulta de dados

O DQL(Data Query Select) é o conjunto SQL que define o comando mais popular da linguagem o select.

DML: Linguagem de manipulação de dados

O DML(Data Manipulation Language) é o subconjunto do SQL que define os comando mais usados para manipular os dados armazenados em um banco. Esse é um dos subconjuntos mais utilizados.

DDL: Linguagem de definição de dados

O DDL(Data Definition Language) é o subconjunto SQL que apresenta comandos usados para gerenciar as estruturas do banco de dados.

DCL: Linguagem de controle de dados

O DCL(Data Control Language) é o subconjunto no qual encontramos comando para controlar o acesso aos dados da nossa base ou banco.

DTL: Linguagem de transação de dados

O DTL(Data Transaction Language) - Também conhecido como TCL(Transaction Control Language) - é o subconjunto do SQL que define comandos que utilizamos quando é necessário gerenciar transações feitas no banco ou base.
