Obtendo dados das tabelas(SELECT)

  • Podemos visualizar dados de uma tabela com o comando select;
  • Quando utilizamos o comando select, estamos prontos para realizar uma busca em uma tabela ou visualizar dados gerais;
  • Assim como usamos select * from nomedatabela para visualizar todas as linhas e colunas, podemos também filtrar a seleção, como ver de forma ordenada;
  • Para isso usamos os parâmetro do comando select;

Operadores de relação

Informamos a relação do campo para a seleção

nome nascimento
Allan Costa 1994-11-10

Alguns dos operadores de relação são:

Operador Relação
in() Em
between-and Entre
or Ou
like Que tenham determinado caractere no valor
> Maior
< Menor
>= Maior ou igual
<= Menor ou igual
!= Diferente

Distinguindo

Fazemos a distinção de valores. Simplificando, é feita a seleção de todos os valores, mas são excluidos os repetidos.

sexo
M
F

Funções de agregação

Podemos somar valores de um campo, mostrar o maior e menor valor registrado ou mostrar a média

count
5

Algumas das funções de agregação são:

Função Agregação
count() Contar
max() Valor máximo
min() Valor mínimo
sum() Somar valores
avg() Média

Agrupando

No agrupamento é feito a separação de valores, e se tivermos repetidos, eles se encaixam no mesmo grupo

sexo count(sexo)
M 2
F 3
