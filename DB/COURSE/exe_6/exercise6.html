Host

  • Host(hospedeiro), é qualquer máquina ou computador conectado a uma rede;
  • Assim podendo oferecer informações, recursos, serviços e aplicações aos usuários ou outros na rede;
  • É o responsavel por implementar a estrutura da camada de rede de endereçamento;
  • Fundamental para o banco de dados, pois é o local de hospedagem;
  • Temos 3 tipos de host, que são: cloud server, servidor dedicado e servidor compartilhado;

Cloud Server

Ele é a famosa nuvem que hoje em dia é muito comum vermos pessoas realizando o armazenamento de seus arquivos em nuvem. E isso não é diferente para as empresas.

O host em nuvem, o cloud server, é caracterizado pelo processamento virtual de suas aplicações, ou seja, o armazenamento de dados é feito por diversas máquinas que permanecem interligadas e executam as mesmas tarefas.

Servidor dedicado

É ideal para os sites que possuem uma grande quantidade de tráfego.

O servidor dedicado é uma computador que te permite acessar dados da mesma forma que a execução de um software remota.

Servidor compartilhado

O servidor compartilhado funciona de maneira semelhante ao dedicado. Contudo, como seu próprio nome indica, um site é hospedado por um grupo de servidores.

Os recursos desses servidores são compartilhados com diferentes sites, blogs e página web de maneira geral.

Esse é um dos tipos de servidores mais utilizados no mundo, justamente pelo seu ótimo custo beneficio.
