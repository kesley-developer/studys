Gerenciando banco de dados pelo terminal

  • Para acessar uma base de dados ou gerenciar uma, é muito comum utilizarmos uma ferramenta como o MYSQL Workbench;
  • Porém, o terminal é muito utilizado e comum para acessar ou gerenciar um banco de dados;

1

Acessamos o local do mysql e acessamos o servidor

c:\xampp\mysql\bin> mysql -u root -h localhost -p

2

Logo em seguida será necessário informar a senha de acesso

Enter password:

3

Sendo assim, temos acesso ao servidor e poderemos manipular a base de dados

MariaDB[(none)]> show databases

4

Caso quisermos usar uma base de dados utilzamos o comand USE

MariaDB[cadaster]> use cadaster

Seguindo o passo a passo, podemos manipular livremente a base de dados. Lembrando que os comandos SQL são os mesmos, porém as devem ser feitos com apenas uma linha no terminal do sistema operacional.
