Tipos primitivos

  • Também conhecido como tipos de dados, são um conjunto de tipo de dados básicos a partir dos quais todos os outros tipos são construidos;
  • Todas as linguagens de programação possuem seus tipos e alguns desses são: números, textos, alfanuméricos, entre outros;
  • Os tipos primitivos da linguagem SQL são os seguintes: numérico, data ou tempo, literal e espacial;

Numérico

  • Inteiro - TinyInt, SmallInt, Int, Mediumint, Bigint
  • Real - Decimal, Float, Double, Real
  • Lógico - Bit, Boolean

Data/Tempo

  • Date, DateTime, TimeStamp, Time, Year

Literal

  • Caractére - Char, VarChar
  • Texto - TinyText, Text, MediumText, LongText
  • Binário - TinyBlob, Blob, MediumBlob, LongBlob
  • Coleção - Enum, Set

Espacial

  • Geometry, Point, Polygon, MultiPolygon

Como podemos observar acima, os tipos primitivos da linguagem SQL se dividem em númerico, data ou tempo, literal e espacial. Dentro desses tipos primitivos, temos diversos tipos de dados.

Esses tipos de dados devem ser informados durante a criação de um campo em uma tabela.
