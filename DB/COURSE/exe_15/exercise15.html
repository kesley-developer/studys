Junção de tabelas(JOIN)

  • Dependendo do tipo de banco de dados, podemos ter mais de uma tabela;
  • Estas tabelas podem se relacionar uma com a outra, mas devemos informar as tabelas que farão a relação e junção, e também definir as chaves estrangeiras
  • Para isso é utilizado o comando join para juntar as tabelas, e para dar sentido a essa seleção usamos o parâmetro on;

O comando join se divide em 3 partes que são:

  • inner join ou join --- Não prioriza a seleção de nenhuma tabela
  • left join --- Prioriza a tabela a esquerda na seleção
  • right join --- Da prioridade a tabela a direita

Definindo Foreign Key(chave estrangeira) de uma tabela

field type null key default extra
id int(11) no PRI null auto_increment
nome varchar(30) no null
profissao varchar(10) yes null
sexo enum('M','F') yes null
nacionalidade varchar(20) yes Brasil
cursodousuario int(11) yes MUL null

PRI - PRIMARY KEY(CHAVE PRIMÁRIA)
MUL - FOREIGN KEY(CHAVE ESTRANGEIRA)

Juntando duas tabelas

Tabela persons(pessoas) Tabela courses(cursos)
id nome idcurso nome ano idusuario
1 Kesley Keven 5 HTML5 2021 1
2 Isabela Silva 3 PYTHON 2020 2
3 Gabriela Costa 9 MYSQL 2018 3

Observando acima, temos uma junção de tabelas, onde a sua relação é formada pela chave primária da tabela persons com a chave estrangeira da outra tabela chamada courses.

Juntando várias tabelas

Para juntar diversas tabelas, criamos uma tabela extra como se fosse um filtro de uma relação, e inserimos as chaves estrangeiras das chaves primarias de outras tabelas dentro dessa tabela extra

Tabela view(visualizar)
field type null key default extra
id int(11) no PRI null auto_increment
data date yes null
idusuario int(11) yes MUL null
idcurso int(11) yes MUL null
  • 1 - Tabela persons
  • 3 - Tabela view
  • 2 - Tabela courses

Sendo assim, podemos iniciar a junção e o relacionamento entre as tabelas
Tabela persons(pessoas) Tabela view(visualizar) Tabela courses(cursos)
id nome id data idusuario idcurso idcurso nome ano
1 Kesley Keven 1 2021-10-09 1 5 5 HTML5 2021
2 Isabela Silva 2 2020-12-30 2 3 3 PYTHON 2020
3 Gabriela Costa 3 2019-10-01 3 9 9 MYSQL 2018

A tabela view cria o relacionamento e mostra os registros da tabela persons, que se relacionam com a tabela courses através das chaves.

Aplicando o JOIN e o parâmetro ON corretamente, podemos selecionar e visualizar os registros que se relacionam um com o outro.
