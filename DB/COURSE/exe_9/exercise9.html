Alterando a estrutura da tabela(ALTER TABLE e DROP TABLE)

  • Ao inserir dados a tabela, podemos modificar sua estrutura;
  • Como alterar um campo ou elimina-lo;
  • Para efetura isso utilizamos alter table e drop table;
  • A funcionalidade do alter table serve para alterar a estrutura de uma table, ou seja, com ele podemos adicionar um novo campo a nossa tabela;
  • O drop table é usado para eliminar um campo ou coluna da tabela;

Adicionando campo

field type null key default extra
id int(11) NO PRI null auto_increment
nome varchar(20) NO null
profissao varchar(30) YES null

Como podemos interpretar acima, mesmo o campo profissao não sendo existente durante a criação de uma tabela, pode ser inserido depois com o comando alter table.

Com o comando alter table, podemos modificar colunas, renomear ou alterar o nome de uma tabela. Caso quisermos eliminar a tabela usamos o comando drop table.
