Histórico e versões - git log e git show

  • Ao efetuarmos um commit, informações são geradas;
  • Podemos visualizar essas informações com os comandos git log e git show;

git log

  • Com o comando git log podemos visualizar o log completo do commit do git;
  • O log do commit lista as seguintes informações:
    • Nome do commit(via hash);
    • Autor;
    • Email;
    • Data;
    • Descrição;

git log

Com git log podemos visualizar informações de cada commit feito no repositório, mas também podemos especificar o arquivo de qual queremos informações com git log "nomedoarquivo.js"

git show

  • Podemos visualizar as alterações realizadas nos arquivos de um projeto, com detalhes, utilizando o comando git show "nomedocommit";
  • O nome do commit pode ser obtido com o comando git log, e basta usar a partir dos quatro primeiros caracteres para realizar a consulta;
  • Se o nome do commit não for fornecido, serão mostrados detalhes do último arquivo consolidado;

git show ae4f

Com o exemplo acima, fornecemos o nome do git juntamente com o comando git show. Com isso serão mostrados informações e alterações feitas ao commit especificado.
