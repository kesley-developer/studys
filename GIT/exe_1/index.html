Controle de Versão(VC)

Controle de Versão(VC) trata do gerenciamento de versões de um projeto, incluindo documentos, código-fonte, programas, etc...

Por quê precisamos de controle de versão?

Tarefas comuns a trabalhos de criação de conteúdo:

  • Criar;
  • Salvar;
  • Editar;
  • Salvar novamente;
  • Abrir para revisão;
  • Editar novamente;
  • Salvar outra vez;
  • Comparar arquivos;
  • Compartilhar com colegas;
  • Colegas editam;
  • Colegas salvam;

Sendo assim, se criarmos um código ou outro tipo de arquivo e quisermos edita-lo usamos o controle de versão para salvar suas versões editadas.
