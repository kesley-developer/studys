Removendo completamente as mensagens de commit

  • Ao fazer um commit, inserimos uma mensagem indicando o que foi alterado no arquivo ou informações básicas;
  • Podemos remover todas as mensagens de um commit;

git filter-branch --msg-filter "echo ''" -- --all

Efetuando o comando acima, todas as mensagens de commit, serão removidas e substituídas por uma string vázia, ou seja, por uma mensagem sem valor.

Caso desejarmos enviar estas alterações realizadas ao repositório remoto do GitHub, prosseguimos com o seguinte comando:

git push --force --all

Feito isso, o repositório do GitHub será atualizado com as definições que informamos.
