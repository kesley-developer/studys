Comparando alterações nos arquivos - git diff

  • Podemos verificar as alterações realizadas nos arquivos no diretórios de trabalho(Cópia local) com o comando git diff;

git diff

  • O comando git diff revisa alterações atuais nos arquivos;
  • Lembrando que git diff é o mesmo que git diff HEAD;

git diff

Com git diff podemos ver o que foi adicionado ou removido nos arquivos ou seja, veremos uma versão anterior e atual do arquivo, lembrando que também podemos especificar o arquivo utilizando git diff "nomedoarquivo".

Vale ressaltar que o comando git diff mostra apenas os arquivos que não estão na Staging area(Estado preparado), mas caso quisermos visualizar os arquivos que estão prontos para commit usamos o comando git diff --staged.

git diff --staged
