Removendo arquivos do staging area

  • Como já sabemos, o staging area é o estado consolidado e etapa onde os arquivos estarão prontos para um commit;
  • Podemos remover arquivos da área de stage utilizando o comando git rm --cached nomedoarquivo.txt;

git rm --cached archive.txt

O comando git rm --cached removeu o arquivo do rastreamento ou estado preparado, então caso quisermos realizar um commit teremos que fazer todo preparo novamente.
