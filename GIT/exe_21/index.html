Comando git pull

  • O comando git pull é usado para buscar e baixar conteúdo de repositórios remotos, e fazer a atualização imediata ao repositório local para que os conteúdos sejam iguais;
  • Fazer o merge de alterações upstream remotas no repositório local, é algo comum em fluxos de trabalho de colaboração baseados em git, simplificando o trabalho em equipe;

git pull git@github.com:kesley-dev/project.git → Podemos trocar pela origin caso já esteja configurada

Com o comando git pull, serão baixadas todas as atualizações do diretório do projeto em nossa máquina local, ou seja, novos arquivos e edições.
