Excluindo e restaurando arquivos - git rm e git checkout

  • Para remover um arquivo podemos simplesmente utilizar o gerenciador de arquivos de nosso sistema operacional;
  • Mas é importante utilizarmos comandos dentro do git para excluir ou restaurar arquivos, pois assim podemos rastrear as informações do projeto;
  • Para isso usamos os comandos git rm e git checkout;

git rm

  • Este comando remove um determinado arquivo;

git rm archive1.txt

Para o comando git rm funcionar, primeiro devemos realizar um commit, sendo assim a exclusão do arquivo será bem sucedida.

git log --diff-filter=D --summary

  • Este comando mostra os arquivos de commit que foram deletados;

git log --diff-filter=D --summary

Com este comando, temos informações de arquivos que foram removidos do commit.

git checkout

  • Com git checkout podemos restaurar arquivos removidos;
  • Basta pegarmos o nome do commit com o comando git log --diff-filter=D --summary e informamos o nome do arquivo;

git checkout a10f~1 archive.txt

Assim podemos restaurar os arquivos excluidos de um commit, utilizando git checkout.
