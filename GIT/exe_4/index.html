Fluxo de operações básico em um VCS

  • MASTER
  • BRANCH
  • BRANCH
  • MERGE
  • MASTER

MASTER - Inicio do projeto ou arquivo inicial

BRANCH - Operações feita ao projeto ou ramificação

MERGE - Significa fundir as operações e transformar o projeto inicial em uma nova versão

Acima temos um exemplo básico do fluxo de operação de um versionamento, por tanto a partir do projeto inicial(MASTER) teremos operações(BRANCH) sendo feitas, após termino dessas operações será fundido(MERGE) ao projeto ou arquivo de inicio a sua nova versão ou implementação.
