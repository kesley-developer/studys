Obtendo ajuda com o comando help

  • Podemos obter ajuda e instruções de comando do git;
  • Para isso, usamos o comando git help;

git help

Assim teremos informações e instruções de uso de alguns comandos do git, mas caso quisermos visualizar informações mais complexas usamos git help -a ou git help -g.

Também podemos obter ajuda informando somente o comand desejado, por exemplo:

git help git commit
