Clonando repositório remoto - git clone

  • Podemos clonar um repositório do GitHub na nossa máquina local;
  • Uma clonagem seria uma cópia do projeto com todos os arquivos;
  • Para efetuar isso, utilizamos o comando git clone;

git clone git@github.com:Kesley-dev/project.git

Efetuando o git clone e informando o endereço SSH do repositório, teremos uma cópia do repositório em nossa máquina.

Lembrando, se caso quisermos mudar o nome do repositório que será clonado, informamos o novo nome no final do comando.

Vale reforçar que quando trabalhamos em equipe em um emsmo projeto, cada desenvolvedor deverá clonar o repositório. Está é a principal função do git clone.
