Atributos do input

O elemento INPUT é muito utilizado, principalmente na construção de formulários no HTML.

Atributo Função
id Onde colocamos o identificador de cada input
name Onde defininmos o nome de cada input
placeholder Usado para inserir um texto no fundo do input
maxlength Onde informamos a quantidade máxima de caractéres
minlength Informamos para definir a quantidade mínima de caractéres
required Indica que o campo é de preenchimento obrigatório
readonly Especifica que o campo é de somente leitura
disabled Desabilita o campo
value Onde fica o valor digitado no input
autofocus Coloca o foco no input assim que a página é carregada
type Atributo obrigatório para especificar o tipo de input
