Grid CSS

Grid é um recurso do CSS que permite desenvolver layouts em grid, e ele é um valor da propriedade display do CSS. É utilizado em containers que contém itens e este container controla a disposição dos itens, mas os itens também podem alterar o seu posicionamento. Os grids são divididos em colunas e linhas. Lembrando que podemos inserir um espaçamento entre as áreas com recurso de GAP.
