Atributos de elementos

Atributos HTML são palavras especiais usadas dentro da tag de abertura para controlar o comportamento do elemento. Os atributos HTML são um modificador de um tipo de elemento HTML. Com eles podemos identificar melhor um elemento, informar qual arquivo aquela tag deve utilizar, indicar o tipo de um campo de texto, etc...

Há dois tipos de atributos no HTML, os globais que são aceitos por todas as tags, como por exemplo: class, id, lang, style e algumas outras, também existem os específicos que somente algumas tags possuem, como src, disabled, href, label, entre outras.

class

Classes são como classificações de uma tag/elemento, para que no CSS possamos estilizar uma tag específica, ou um conjunto de tags. Também é possivel usar no JavaScript para selecionar uma tag específica.

h1 class="title">Titulo da página h1

id

Identificar de forma única um elemento naquela página HTML. É utilizado para identificar destino de âncoras, labels e outras funcionalidades neste sentido.

div id="menu">

src

Comumente utilizado para indicar para a tag qual arquivo ou mídia utilizar. Recebe valores como links (https://www.google.com/flowers.jpeg) ou o nome de um arquivo já presente no projeto (imgs/flowers.jpeg).

img src="https://www.google.com/flowers.png">

alt

Este atributo fornece informações alternativas para uma imagem se um usuário por algum motivo não puder visualizá-la (devido a conexão lente, um erro no atributo src ou se o usuário usar o leitor de tela).

img src="https://www.google.com/flowers.jpeg" alt="flowers images">

href

Para a tag "a" o atributo href especifica a url da página para a qual o link vai.

a href="www.google.com">Clique para acessar o google a

lang

O atributo lang especifica o idioma do conteúdo da tag. Os exemplos comuns são "en" para inglês, "es" para espanhol, "fr" para francês e assim por diante.

html lang="en">

target

Esse atributo abre o link do documento em uma nova janela ou aba.

a href="www.google.com" target="blank">Clique para acessar o google a
