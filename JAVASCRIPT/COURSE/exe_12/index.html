Comandos break e continue

  • Com break podemos encerrar uma instrução;
  • Com o continue podemos pular uma instrução;
  • São utilizados na maioria das vezes em estruturas de repetição(while e for), ou seja em loops;

break

  • Pode quebrar uma estrutura de repetição, simplificando, o comando break encerra a execução de um loop;

continue

  • Caso o comando continue seja usado em uma estrutura de repetição, o loop não será encerrado, mas quando o loop atingir tal condição pode-se ignorar tal valor ou adicionar algo durante a execução;
