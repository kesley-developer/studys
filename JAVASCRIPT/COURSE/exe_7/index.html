Operadores condicionais

  • São instruções que podem determinar o fluxo de uma aplicação;
  • Amplamente utilizados, deixam os softwares dinâmicos;
  • Exemplos são: if,else if, else;

if

  • Executa um bloco de código se a instrução for true(verdadeira);
  • Se for false(falsa), segue o código normalmente;
  • Lembrando que: true e false são um tipo de dado boolean;

else if

  • O else if tem a possibilidade de fazer outra verificação e adicionar mais um bloco de código;
  • Caso o if seja falso, o bloco de código do else if será executado caso a sua instrução seja válida;

else

  • Caso a instrução do if e else if for negativa, podemos adicionar a condicional else;
  • Não passamos instruções na condição else, portanto ela será executada de qualquer modo se as outras condicionais if e else if não forem válidas;
