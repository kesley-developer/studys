Funções(function)

  • Funções são bloco de códigos reutilizáveis;
  • Simplificando, evitamos a repetição da lógica de um programa em diversas partes do código;
  • A função precisa ser invocada para ser executada;
  • Cada linguagem de programação possui suas funções, mas podemos tambem criar uma função, ou seja personalizar;
  • Em JavaScript declaramos uma função com function;
