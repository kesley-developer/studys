Particularidades do JavaScript

  • Tipagem fraca, ou seja, as váriaveis não possuem seu tipo númericos ou de strings, por isso é possivel altera-las durante a construção e execução do código;
  • O ponto e vírgula é opcional a cada instrução(linha de código), portanto é recomendado o seu uso para evitar possiveis bugs;
  • Há comentários para uma linha e múltiplas linhas de código, simplificando, podemos comentar não somente uma linhas de código, mas sim várias;
