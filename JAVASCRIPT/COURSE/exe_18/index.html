Métodos de string

  • O objeto string também possui métodos muito reais;
  • Estes métodos auxiliam a manusear textos em nosso software;

length

  • Analisa o tamanho da string;
  • Podemos utilizar o length para medir o tamanho de arrays;

indexOf

  • Mostra em qual posição um certo caractére de uma string aparece pela primeira vez;

slice

  • Extraí a palavra de uma string;
  • Para aplicar este método é preciso informar a posição inicial e final a ser extraída;

replace

  • Substitui um caractére de uma string;
  • Para o método ser aplicado é preciso informar a palavra a ser substituida e a palavra a ser inclusa;

toLowerCase

  • Deixa o texto com letra minúscula;

toUpperCase

  • Deixa o texto com letra maiúscula;

trim

  • Elimina os espaçõs laterais de um texto;

split

  • Transforma elementos de uma string em um array;

lastindexOf

  • Ao contrário do indexOf, o método lastindexOf mostra em qual posição do índice um certo caractére aparece pela última vez;
