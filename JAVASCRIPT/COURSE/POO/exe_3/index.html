Método constructor

  • Em JS praticamente qualquer tipo de dado é um objeto e cada item dessa "coleção de valores", é chamado de propriedade;
  • Cada propriedade é composta por um par de "nome: valor", e quando uma propriedade armazena uma função, ela se torna o que chamamos de método;
  • O constructor é um método padrão em JS, que é relacionado a classe, ou seja quando um novo objeto for construído passamos as informações a serem iniciadas dentro do método constructor;
