Classes

  • As classes são um modelo para criar objetos;
  • Elas encapsulam dados com código para trabalhar esses dados;
  • Para declarar uma classe usamos class e em seguida o nome sugerido para a classe;
  • Existem outras formas para declaração de um classe que são: atribuir a classe a uma variável, retornar uma classe dentro de uma função e o método tradicional que consiste em criar a classe de forma similar a uma função;
  • As classes são "Funções especiais" que constroem novos objetos;
