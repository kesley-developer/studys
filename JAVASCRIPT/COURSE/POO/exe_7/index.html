Módulos

  • Resumidamente os módulos em JavaScript é uma forma de compartilhar informações entre arquivos;
  • Dessa forma, todos os arquivos que realizarem a importação desse módulo poderão utilizar e usufruir de suas funções ou informações que foram expostas;
  • Utilizamos export para informar que algo será exportado e import para importar este algo;
  • Para efetuar a modularização, é preciso criar um pacote que é arquivo chamado package.json na pasta do projeto e para isso usamos o terminal ou criamos manualmente;

Exemplo do pacote

package.json

{

"name": "register-persons", - NOME DO PACOTE

"version": "1.0.0", - VERSÃO

"description": "Registrando pessoas na classe", - DESCRIÇÃO DO PACOTE

"main": ".js/main.js", - CAMINHO E TIPO DE ARQUIVO

"scripts": {

"test": "echo \" error: no test specified\" && exit1"

}, - COMANDO DE TESTE

"author": "Kesley Keven", - AUTOR

"license": "ISC", - LICENÇA

"type": "module" - TIPO DE PACOTE

}
