Métodos acessores get e set

  • Os métodos acessores nos permitem acessar o nome de uma propriedade ou chaves de um objeto;
  • Com este método podemos acessar o valor de um atributo ou mudar este valor;
  • O set configura e defini dados de um atributo;
  • O get pega dados de um atributo ou extrai valores do mesmo;
