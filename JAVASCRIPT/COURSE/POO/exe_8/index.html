Padrão MVC(Model View Controller)

  • MVC funciona como um padrão de arquitetura de software;
  • Seguindo esse padrão temos uma melhora de conexão entre as camadas de dados, lógica de negócio e interação com usuário;
  • Através da sua divisão em três componentes Model, View e Controller, o processo de programação se torna algo mais simples e dinâmico;

  • Model - Modelo de dados

  • View - Visualização dos dados

  • Controller - Controlador de ações
