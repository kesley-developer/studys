Métodos de array

  • Os arrays também possuem métodos;
  • Facilitando o código para: adicionar e remover elementos, resgatar apenas uma parte do array e etc;

push

  • Adiciona elementos ao array;
  • O elemento é adicionado ao final do array;

pop

  • Remove o elemento localizado no último indíce do array;

unshift

  • Adiciona elementos no primeiro indíce do array;

shift

  • Remove o elemento do primeiro indíce do array;

isArray

  • Verifica se é um array, caso não seja retornará false;
  • É preciso informar o objeto para seu funcionamento que no caso é Array;

splice

  • Adiciona ou remove um elemento de determinada posição do array;
  • Para o funcionamento do método é preciso informar 3 parâmetros, a posição do índice, a quantidade de caractéres a ser excluido e o valor a ser adicionado;

join

  • Transforma o array em uma string;

reverse

  • Transforma a ordem do array em reversa
