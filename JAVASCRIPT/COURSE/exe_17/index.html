Métodos numéricos

  • O objeto Number, pai dos números, contém métodos muito úteis para trabalhar com números em JS;
  • A maioria dos tipos de dados também tem um objeto pai, como: String, Object e Array;

parseFloat

  • Transforma um valor inteiro em flutuante;

parseInt

  • Transforma um valor flutuante em inteiro;

toFixed

  • Limita a quantidade de casas decimais de um valor;

isNaN

  • Verifica se o valor não é um número, caso seja retornará false;

MAX_VALUE e MIN_VALUE

  • Valores máximos e minimos que o JavaScript reconhece;
  • Qualquer outro valor fora dos limites do JS será tratado como infinity;
