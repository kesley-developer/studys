Math

  • Math é um objeto embutido que tem propriedades e métodos para constantes e funções matemáticas, mas não é um objeto de função;
  • Podemos gerar números aleatórios com math e fazer calculos complexos;
  • Podemos não só realizar cálculos mas também acessar índices de forma aleatória com o método random() em math;

Math.floor()

  • Retorna o maior valor inteiro;

Math.min()

  • Identifica e retorna o menor valor dentre os parâmetros recebidos;

Math.max()

  • Identifica e retorna o maior valor dentre os parâmetros recebidos;

Math.random()

  • Retorna um valor aleatório flutuante entre 0 e 1;
