Operadores de comparação

  • São comumente utilizados para verificação nas estruturas de condição(if,else if, else);
  • Temos os seguintes operadores: ==(igual a), !=(diferente de), >(maior que), <(menor que), >=(maior ou igual a), <=(menor ou igual a);
  • A partir de uma expressão de comparação podemos obter um true ou false;
